fs-migrate 文件处理命令行工具

用法： fs-migrate [子命令] [选项]...

启动：
  -h,  --help                          打印此帮助

子命令：
  rename                               重命名文件
                  --sr,--sourceRegex
                                       需要被重命名的部分
                  --rr,--replacementRegex
                                       替换品
